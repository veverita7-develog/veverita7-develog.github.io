## Quick Start

### 클론

```bash
git clone https://github.com/vue-guide/vue-guide.github.io.git
```

### 브랜치 변경(writing)

```bash
git checkout -b writing origin/writing
```

### 설치

```bash
npm install -g hexo

npm install
```



------

## 사용법

### 1. 문서 업로드

`/source/_posts` 경로에 마크다운(`.md`) 파일 업로드

### 2. Deploy 명령어 실행

```bash
npm run deploy
```
